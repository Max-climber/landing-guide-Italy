# Решение проблемы доступа к сайту из России

## Проблема
Сайт не открывается в России без VPN, но на некоторых устройствах работает. Основная аудитория - РФ.

## Причины проблемы

1. **Vercel CDN может быть заблокирован** - Vercel использует CDN, который может быть недоступен в России
2. **Google Fonts заблокированы** - уже исправлено (убраны из index.html)
3. **DNS проблемы** - некоторые DNS серверы в России могут не резолвить домены Vercel

## Решения (по приоритету)

### 1. ✅ Использовать российский хостинг (РЕКОМЕНДУЕТСЯ)

**Варианты:**
- **Timeweb** (timeweb.com) - хорошая поддержка, доступные цены
- **REG.RU** (reg.ru) - популярный российский хостинг
- **Beget** (beget.com) - надежный хостинг с хорошей поддержкой
- **Selectel** (selectel.com) - для более серьезных проектов

**Инструкция для Timeweb:**
1. Зарегистрируйтесь на timeweb.com
2. Закажите хостинг (можно начать с минимального тарифа)
3. Загрузите файлы из папки `dist` через FTP или файловый менеджер
4. Настройте домен (если есть) или используйте поддомен timeweb.ru

### 2. ✅ Использовать Cloudflare Pages (АЛЬТЕРНАТИВА)

Cloudflare Pages лучше работает в России, чем Vercel:
1. Зарегистрируйтесь на cloudflare.com
2. Подключите GitHub репозиторий
3. Настройки:
   - Build command: `npm run build`
   - Build output directory: `dist`
   - Framework preset: Vite

### 3. ✅ Настроить зеркало на российском домене

Если основной домен на Vercel, можно создать зеркало:
1. Зарегистрируйте домен в зоне .ru или .рф
2. Настройте хостинг на российском провайдере
3. Настройте автоматическую синхронизацию с основным сайтом

### 4. ✅ Использовать CDN с российскими серверами

- **Cloudflare** (бесплатный план) - имеет серверы в России
- **KeyCDN** - международный CDN с хорошей поддержкой России
- **CDN77** - европейский CDN, работает в России

### 5. ✅ Настроить DNS

Проблема может быть в DNS. Рекомендуется:
- Использовать публичные DNS: 8.8.8.8 (Google) или 1.1.1.1 (Cloudflare)
- Или российские: 77.88.8.8 (Yandex DNS) или 178.176.1.1 (Mail.ru)

## Что уже исправлено

✅ Убраны Google Fonts из index.html (они блокируются в России)
✅ Используются только локальные шрифты (подключены через @font-face в index.css)

## Рекомендации

**Для быстрого решения:**
1. Зарегистрируйтесь на Timeweb или REG.RU
2. Загрузите файлы из папки `dist` на хостинг
3. Настройте домен

**Для долгосрочного решения:**
1. Используйте Cloudflare Pages вместо Vercel
2. Настройте автоматический деплой из GitHub
3. Добавьте российский домен как зеркало

## Проверка доступности

После настройки проверьте доступность:
- https://www.blocked.org.uk/ - проверка блокировок
- https://2ip.ru/ - проверка доступности из России
- Попросите друзей/коллег в России проверить доступность
