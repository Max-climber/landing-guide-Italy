# Инструкция по добавлению изображений курортов

## Структура папок

Изображения для каждого курорта хранятся в отдельных папках:

```
public/images/resorts/
├── piani-di-bobbio/
│   ├── 1.jpg
│   ├── 2.jpg
│   ├── 3.jpg
│   └── ...
├── madesimo/
│   ├── 1.jpg
│   ├── 2.jpg
│   └── ...
├── sankt-moritz/
│   ├── 1.jpg
│   └── ...
├── bivio/
├── valmalenco/
├── aprica/
└── livigno/
```

## Правила именования

1. **Формат файлов**: только `.jpg` (можно использовать `.jpeg`, но рекомендуется `.jpg`)
2. **Именование**: изображения должны называться `1.jpg`, `2.jpg`, `3.jpg` и так далее (начиная с 1)
3. **Количество**: рекомендуется 6-8 изображений на каждый курорт
4. **Размер**: изображения могут быть разных размеров, они автоматически подстроятся под размер карточки

## Названия папок курортов

Каждый курорт имеет свою папку с определенным названием:

| Курорт | Название папки |
|--------|---------------|
| Piani di Bobbio | `piani-di-bobbio` |
| Madesimo | `madesimo` |
| Sankt Moritz | `sankt-moritz` |
| Bivio | `bivio` |
| Valmalenco | `valmalenco` |
| Aprica | `aprica` |
| Livigno | `livigno` |

## Как добавить изображения

1. Откройте папку нужного курорта (например, `public/images/resorts/piani-di-bobbio/`)
2. Добавьте изображения с именами `1.jpg`, `2.jpg`, `3.jpg` и т.д.
3. Изображения автоматически появятся в карусели на сайте

## Пример

Для курорта "Piani di Bobbio":
```
public/images/resorts/piani-di-bobbio/
├── 1.jpg  ← Первое изображение (главное)
├── 2.jpg  ← Второе изображение
├── 3.jpg  ← Третье изображение
├── 4.jpg
├── 5.jpg
├── 6.jpg
├── 7.jpg
└── 8.jpg
```

## Важные замечания

- **Порядок важен**: изображения отображаются в порядке их номеров (1, 2, 3...)
- **Первое изображение**: `1.jpg` будет показано первым при загрузке страницы
- **Если изображение отсутствует**: браузер покажет ошибку загрузки. Убедитесь, что все файлы существуют
- **Оптимизация**: рекомендуется оптимизировать изображения перед загрузкой для лучшей производительности

## Технические детали

- Изображения загружаются из папки `public`, поэтому путь в коде: `/images/resorts/[название-папки]/[номер].jpg`
- Карусель автоматически определяет количество изображений (до 8 штук)
- Если изображение не найдено, браузер покажет стандартную ошибку загрузки

## Поддержка форматов

В настоящее время поддерживается только формат `.jpg`. Если нужно добавить поддержку других форматов (`.png`, `.webp`), необходимо изменить функцию `getResortImages` в файле `src/components/Resorts.jsx`.
