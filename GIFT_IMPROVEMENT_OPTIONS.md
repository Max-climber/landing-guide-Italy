# Варианты улучшения качества подарка

## Проблема
Текущее изображение подарка размытое, так как используется маленькое изображение (48x48px) с icons8.com, которое растягивается.

## Реализованное решение
Использовано изображение большего размера (96x96px) с параметром `imageRendering: 'crisp-edges'` для улучшения качества на ретине-дисплеях.

## Альтернативные варианты (если текущее решение не подходит):

### Вариант 1: SVG подарок (рекомендуется)
Создать собственный SVG подарка - будет четким на любом разрешении.
- Преимущества: идеальное качество, маленький размер файла, легко кастомизировать
- Недостатки: нужно создать SVG

### Вариант 2: Локальное изображение высокого разрешения
Добавить изображение подарка в `public/images/` с разрешением минимум 256x256px
- Преимущества: полный контроль над качеством
- Недостатки: больший размер файла

### Вариант 3: Использовать другой источник иконок
Например, использовать иконки из:
- Heroicons (SVG)
- Font Awesome (векторные)
- Material Icons (векторные)
- Или создать собственную иконку

### Вариант 4: Увеличить размер до 128x128 или 256x256
Использовать еще большее изображение с icons8.com:
```jsx
src="https://img.icons8.com/emoji/128/wrapped-gift.png"
// или
src="https://img.icons8.com/emoji/256/wrapped-gift.png"
```

## Текущая реализация
Используется вариант с изображением 96x96px с улучшенным рендерингом.
