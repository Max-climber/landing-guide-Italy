# La Vacanza Bianca - Премиальные горнолыжные туры в Италию

Одностраничный лендинг для компании, организующей горнолыжные туры в Италию с индивидуальным сопровождением.

## Технологии

- React 19
- Vite
- Tailwind CSS
- Framer Motion (анимации)
- React Intersection Observer

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Особенности

- Премиальный дизайн с элегантными анимациями
- Параллакс-эффекты при скролле
- Интерактивные карты курортов
- Адаптивный дизайн для всех устройств
- SEO-оптимизация
- Интеграция с WhatsApp для быстрой связи

## Структура проекта

```
src/
  components/
    Navigation.jsx    # Навигация с фиксированным хедером
    Hero.jsx          # Главный экран с параллакс-эффектом
    About.jsx         # О компании
    Programs.jsx      # Три программы обслуживания
    Resorts.jsx       # Карточки курортов
    Contact.jsx       # Форма обратной связи
    Footer.jsx        # Футер
  App.jsx
  main.jsx
  index.css
```
